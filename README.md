# Flight Radar

A project showing flight information in the region given the latitudes and longitudes in the world.<br/>
In this project, flights and flight information over Turkey are shown.

# Libraries

- @reduxjs/toolkit
- react-redux
- axios
- leaflet
- react-leaflet
- react-paginate
- bootstrap
